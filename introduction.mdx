## Welcome to Expresiones

Expresiones is a complete compiler and interpreter implementation built with ANTLR4, designed to demonstrate fundamental compiler construction principles including lexical analysis, syntax parsing, semantic analysis, and expression evaluation.

The language supports variable declarations with type checking, arithmetic expressions with proper operator precedence, conditional control flow, and comprehensive logical operations.

## Quick Example

Here's a simple program written in the Expresiones language:

```text example.txt
program {
    int x = 10;
    int y = 5;
    int resultado = 0;
    
    // Arithmetic with precedence
    resultado = x + y * 2;
    
    // Conditional logic
    if (resultado > 15) {
        x = x + 100;
    } else {
        x = 0;
    }
}
```

## How It Works

The compiler processes source code through several stages:

<Steps>
  <Step title="Lexical Analysis">
    The **ExpresionesLexer** tokenizes the input source code into a stream of tokens based on the grammar rules.
  </Step>
  <Step title="Syntax Parsing">
    The **ExpresionesParser** analyzes the token stream and builds an Abstract Syntax Tree (AST) according to the grammar's syntactic rules.
  </Step>
  <Step title="Semantic Analysis">
    The **Visitor** traverses the AST, performing type checking, variable scope validation, and building the symbol table.
  </Step>
  <Step title="Execution">
    The visitor evaluates expressions and executes statements, producing output and maintaining program state.
  </Step>
</Steps>
